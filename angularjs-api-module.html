AngularJS Client

The eeh-inengine-api AngularJS module is used to programmatically access the Web API in an AngularJS web application. It is used in the InEngine.NET Dashboard and can be included in 3rd party projects.

It package available via Bower.

bower install eeh-inengine-api --save
